# ansible-nmap-scanner

## Описание
Ansible-плейбук для автоматического сканирования порта 80 на нескольких хостах с помощью NMAP.

## Требования
- Ansible >= 2.9
- Python 3.x
- Доступ по SSH к целевому хосту
- Права sudo на целевом хосте

## Структура проекта

ansible-nmap-scanner/
├── README.md # Документация
├── inventory.ini # Целевой хост
├── nmap_scan.yml # Основной плейбук
└── targets.txt # Список целей для сканирования
